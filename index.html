About Me

This was no easy feat; I quit my job of three years and dedicated three months, 60 hours per week, studying and honing my skills as a coder in the basement of The American Tobacco Campus. I emerged completely enamored with programming, and am so incredibly excited to explore the many doors of the programming world that have been opened to me.

In my free time, I love going to concerts, exploring the culinary scene in the Triangle, taking in a Bulls game, watching a football game or four, or imbibing a nice craft brew with friends.

Fanta-Sci

Fanta-Sci is a fantasy football analytics tool which utilizes object-oriented programming and machine learning to predict projected fantasy football scores. Data is obtained from profootballapi.com, models are instantiated, and then run through the predictor algorithm with the training data tweaked by the user. This app was my final project for The Iron Yard, and I was primarily responsible for the back-end, contacting the API, creating models, saving to the database, calculating predictions based on projections from the machine learning component, etc. Two classmates and I built the app from scratch during a three-week sprint at the end of May and into June. Fanta-Sci utilizes the Django framework and incorporates some JavaScript, Ajax, and JQuery code on the user-end.

  • GitHub Repo: Fanta-Sci (Group Repo)
  • Date: June 2016
  • Client: The Iron Yard - Durham
  • Technologies Utilized: Python, SQL, Django, JavaScript, JQuery, HTML/CSS, NumPy

Text Classifier

Text Classifier was a group project at The Iron Yard in Durham, NC. Our group built a Django app which would take in a block of text, train a Naive Bayes model, and then be able to accurately predict other blocks of text from a similar language. We built and tested it using European languages. This project utilized Scikit-learn to make the predictions.

  • GitHub Repo: Text Classifier (Group Repo)
  • Date: May 2016
  • Client: The Iron Yard - Durham
  • Technologies Utilized: Python, Django, HTML/CSS, Gunicorn, Scikit-Learn

Traffic Simulator

The Traffic Simulator was an earlier project at The Iron Yard in Durham, NC. It was one of the projects I enjoyed building the most and was, for me, the point where object-orient programming "clicked." The objective was to build a simulation of a number of cars bound by certain rules (top speed, acceleration rate, with a random number of cars slowing down each "second" of the simulation) driving in a continuous loop around a track. Results were graphed out using MatPlotLib and Seaborn in a Jupyter Notebook located on the linked GitHub repository.

  • GitHub Repo: Traffic Simulator (Group Repo)
  • Date: March 2016
  • Client: The Iron Yard - Durham
  • Technologies Utilized: Python, NumPy, SciPy, Pandas, Seaborn, MatPlotLib
